# База знаний барного ассистента

Эта папка содержит TXT файлы с базой знаний для RAG системы.

## Формат файлов

### Формат с чанками (рекомендуемый):

```
[CHUNK]
id: COCKTAIL_001_NEGRONI
tags: коктейли, классические, биттер
keywords: негрони, кампари, джин, вермут
text: Негрони - классический итальянский коктейль.
Ингредиенты:
- 30 мл джина
- 30 мл красного вермута
- 30 мл Кампари
Метод: билд в олд фэшн стакане со льдом
Гарнир: долька апельсина

[CHUNK]
id: COCKTAIL_002_MARTINI
tags: коктейли, классические, джин
keywords: мартини, джин, вермут, оливка
text: Мартини - король коктейлей...
```

### Обычный текстовый формат:

Файлы без разметки [CHUNK] будут автоматически разбиты на параграфы.

## Категории файлов

- `cocktails_*.txt` - рецепты коктейлей
- `beer_*.txt` - информация о пиве
- `spirits_*.txt` - крепкие напитки
- `wine_*.txt` - вина
- `techniques_*.txt` - барные техники
- `equipment_*.txt` - оборудование

Поместите ваши TXT файлы в эту папку и запустите `/reindex` в боте для обновления базы знаний.
