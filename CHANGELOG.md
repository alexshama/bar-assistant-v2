Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-15

### Добавлено
- 🤖 **Базовая функциональность бота**
  - Обработка текстовых, голосовых сообщений и изображений
  - Команды `/start`, `/help`, `/mode`
  - Система пользовательских настроек

- 🧠 **RAG система**
  - JSON-based векторная база данных
  - Поиск по 200+ чанкам знаний о коктейлях и напитках
  - Умная маршрутизация запросов (рецепты, изображения, общие вопросы)
  - Обработка склонений русских слов ("маргариты" → "маргарита")

- 🖼 **Генерация изображений**
  - Интеграция с Gemini 2.5 Flash Image Preview через OpenRouter
  - Автоматическое определение цвета коктейлей по ингредиентам
  - Специальные промпты для сложных коктейлей (B-52, слоистые)
  - Система кэширования изображений для экономии API вызовов

- 🎤 **Голосовое взаимодействие**
  - Speech-to-Text через OpenAI Whisper
  - Text-to-Speech для голосовых ответов
  - Fallback система (OpenRouter + OpenAI)
  - Увеличенные таймауты для стабильности

- 📊 **Административные функции**
  - `/cache_stats` - статистика кэша изображений
  - `/clear_cache` - очистка кэша (только для администраторов)
  - Автоматическая очистка дубликатов в кэше

- 📚 **База знаний**
  - 100+ рецептов коктейлей
  - Информация о различных видах алкоголя
  - Стили пива и их характеристики
  - Техники приготовления и подачи

### Технические особенности
- **Умная маршрутизация**: автоматическое определение типа запроса
  - Вопросы-рассуждения → OpenAI GPT-4
  - Фактические запросы → RAG система
  - Запросы изображений → генерация через Gemini
- **Кэширование**: экономия API вызовов через локальное хранение изображений
- **Обработка ошибок**: graceful fallback между различными API
- **Логирование**: подробные логи для отладки и мониторинга

### Поддерживаемые форматы
- **Текстовые запросы**: рецепты, вопросы, рекомендации
- **Голосовые сообщения**: распознавание и синтез речи
- **Изображения**: анализ фотографий напитков (планируется)

### API интеграции
- **OpenAI**: GPT-4 для чата, Whisper для STT, TTS для голоса
- **OpenRouter**: Gemini 2.5 Flash Image Preview для генерации изображений
- **Telegram Bot API**: через aiogram 3.x
